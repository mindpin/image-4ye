# Image::4ye

Uploading and retrieving images on img.4ye.me.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'image-4ye', git: 'https://github.com/mindpin/image-4ye.git'
```

And then execute:

    $ bundle
